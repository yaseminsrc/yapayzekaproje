# **App Name**: Güzellik Randevu

## Core Features:

- Homepage with Hero: Display the salon name "Güzellik Stüdyo" prominently along with a hero image representing the salon's ambiance.
- Services Page: List services offered: Saç Kesimi (150₺), Manikür (80₺), Cilt Bakımı (200₺), Boya (250₺).
- Pricing Display: Show a dedicated page with a detailed service list and their respective prices.
- Gallery Showcase: Showcase a gallery page with 4 placeholder images, demonstrating the salon's style and work.
- Appointment Form: Implement a form with fields for Ad, Telefon, Email, Hizmet Türü (dropdown), Tarih, Saat.
- Webhook Submission: Upon form submission, POST the data to [N8N WEBHOOK URL] with JSON body: { "ad", "telefon", "email", "hizmet", "tarih", "saat" }.

## Style Guidelines:

- Primary color: Deep pink (#D44DB5) to embody femininity and elegance.
- Background color: Soft lavender (#F3E5F5), a gentle backdrop that complements the primary color.
- Accent color: Orchid pink (#E683D2) used for interactive elements.
- Body and headline font: 'Belleza', a humanist sans-serif with personality suited to fashion, art, and design.
- Use simple, elegant icons for services and navigation.
- Clean, spacious layout with a focus on imagery and readability.
- Subtle transition animations for page navigation and form interactions.
